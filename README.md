# My-App-Blog
Setup framework with Django for my blog of my website.

### Activate virtual environment:
```
vir_env/Scripts/activate
```

### Leave virtual environment:
```
deactivate
```

### Push to Heroku:
```
git add .
git commit -am "make it better"
git push heroku
```

### Push to GitHub:
```
git add .
git commit -m "Add existing file"
git push origin
```
